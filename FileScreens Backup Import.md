# filescreens backup

Realiza backup e importa as configurações do filescreens do serviço de servidor de arquivos no Windows Server.

## Como Usar

Rode o script de dentro de um prompt ou PowerShell elevado como administrador.

**Exemplo:**
```powershell
./filescreens_backup.vbs
```

Serão criados arquivos com o backup das configurações e também o `.bat` para importação.
